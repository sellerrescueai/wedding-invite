# Wedding Invitation Website for Dr. Neha & Pranit

Welcome to your luxurious, custom-built wedding invitation website repository! This is designed to be highly creative, emotional, and 100% free using HTML, CSS, and vanilla JS.

## How to Customize

### 1. Add your Photos (Gallery)
- The images are located in the `images` folder.
- Simply drop your actual photos into `images/` (e.g., `images/photo1.jpg`, `images/photo2.jpg`).
- If you change the filenames, make sure to update the `galleryImages` list in `js/config.js`.

### 2. Configure RSVP (WhatsApp Integration)
- Open the file `js/config.js` in any text editor.
- Look for `whatsappNumber: "9529822207",` and update it if needed. (Include the country code but do not use spaces or `+`).
- You can also modify the `rsvpMessage` text in the same file.

### 3. Add Background Music
- Place an audio file into the `music/` folder and name it EXACTLY `background.mp3` (so the path is `music/background.mp3`).
- For example, you can use your "Save the Last Dance for Me" instrumental version here!

### 4. How to Test locally
- Simply locate the folder on your computer.
- Double-click the `index.html` file.
- It will open in your default browser (Chrome, Edge, Safari, etc.) so you can view it directly!
- **Note:** Browsers sometimes block auto-playing music or script execution of local files. To fully test standard functionality, you may need a local server (like the VS Code "Live Server" extension). But just opening the file works for design feedback!

### 5. How to Publish for Free (GitHub Pages)
1. Go to [GitHub.com](https://github.com/) and create a free account.
2. Create a new repository (name it something like `wedding-invite`).
3. Upload all the files inside the `wedding-invite` folder to this new repository.
4. Go to the repository **Settings** -> **Pages**.
5. Under "Source", select the `main` branch and save.
6. GitHub will give you a public URL (e.g., `https://yourusername.github.io/wedding-invite`) that you can share with your family and friends completely free of charge!
